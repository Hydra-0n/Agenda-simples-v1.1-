# 📒 Agenda Simples v1.1

Projeto desenvolvido em C com o objetivo de praticar conceitos fundamentais de programação, como structs, funções, manipulação de strings e organização de código.

## 🚀 Funcionalidades

- Cadastro de pessoas
- Listagem de contatos
- Pesquisa por CPF
- Busca por nome
- Edição de registros
- Exclusão de contatos
- Validação de CPF duplicado

## 🛠️ Tecnologias Utilizadas

- C
- GCC
- Git
- GitHub
- VS Code

## 📚 Conceitos Praticados

- Structs
- Funções
- Arrays
- Manipulação de Strings (`strcmp`, `strstr`)
- Estruturas de repetição
- Estruturas condicionais
- Organização de código

## ▶️ Como Executar

Compile o projeto utilizando o GCC:

```bash
gcc agenda_simples_v1_1.c -o agenda.exe
```

Execute o programa:

```bash
./agenda.exe
```

> No Windows:

```bash
.\agenda.exe
```
